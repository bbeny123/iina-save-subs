Save Subtitles panel can be toggled via:
  • Menu: Plugin → Subtitles...
  • Keybind: ⌘ + D (customizable via Preferences)
  • Keybind conflict validation requires IINA restart and media load
  • Some binds may fail silently while appearing "Valid" in Preferences
Accepts negative values (-) for backward shifts:
  • Milliseconds: Any whole number
    Examples: 1500 (1.5s delay), -500 (0.5s early)
  • Time Format: Parsed as HH:MM:SS.MSS
    Examples:
    • 1:20 → 1m 20s
    • -1:05.500 → -1m 5.5s
    • 15.5 → 15.5s
    • .500 → 500ms
  • SRT Only: Supports only external .srt tracks. Internal (embedded) subtitles cannot be extracted or saved
  • Delay Detection: Automatically syncs with IINA's subtitle delay
  • FPS Conversion: Corrects progressive sync drift by adjusting subtitle timings to match video framerate
  • Language Suffix: Auto-detects track language from metadata or filename. Fallback can be set in Preferences
